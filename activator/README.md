# activator


## Run your app

Here is an example of a docker run command:

```
docker run -it \
  -p 8888:8888 \
  rayyildiz/activator
```

To run minimal version:
```
docker run -it \
  -p 8888:8888 \
  rayyildiz/activator:minimal
```
